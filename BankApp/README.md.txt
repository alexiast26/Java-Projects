Java Spring Bank Application

A banking application built with Java Spring, featuring a Java Swing user interface and MySQL database integration.

Features

User Authentication: Register a new account or log in to an existing one.

Account Management: Deposit and withdraw funds securely.

Transaction History: View past transactions in detail.

Money Transfers: Send money to other users within the app.

This application securely stores user and transaction data in a MySQL database, providing reliable account management and tracking.
