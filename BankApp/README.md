# Java Bank Application

A **desktop banking application** built with **Java** using **Java Swing** for the user interface and **MySQL** for database management.  

---

## Features

- **User Authentication:** Register a new account or log in to an existing one.  
- **Account Management:** Deposit and withdraw funds securely.  
- **Transaction History:** View detailed past transactions.  
- **Money Transfers:** Send money to other users within the application.  

---

## Technologies Used

- **Java** – Core programming language for application logic.  
- **Java Swing** – Graphical user interface (GUI) for a desktop banking experience.  
- **MySQL** – Relational database for storing user accounts and transaction data.
