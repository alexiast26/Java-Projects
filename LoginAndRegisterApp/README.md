# Java Login and Register 

A basic **login application** built with **Java** using **Java Swing** for the user interface and **MySQL** for database management. The application was made for a better understanding of how to create a login interface. 

---

## Features

- **User Authentication:** Register a new account or log in to an existing one.

---

## Technologies Used

- **Java** – Core programming language for application logic.  
- **Java Swing** – Graphical user interface (GUI).  
- **MySQL** – Relational database for storing user accounts.
